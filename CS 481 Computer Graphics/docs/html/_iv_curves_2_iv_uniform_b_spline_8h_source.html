12 // Usage notes:
13 // A change here from the other curves. Because the B-spline is an
14 // approximating curve, it doesn't make sense to set times at each position,
15 // since the curve won't pass through them. Instead, we triplicate the first
16 // and last positions to force the curve to pass through them, and set times at
17 // those locations. The time interval will be subdivided equally to create
18 // the time at each knot location.
